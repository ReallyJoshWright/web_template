# Web Template

This is a web template for starting a project. It uses CrowCpp, Asio, and Htmx.
The web server is in C++. This is meant to run on Linux.

## Project Structure
- main.cpp
- templates/
- static/
    - assets/
    - images/
    - css/
    - js/
    - ts/
    - libs/
    - components/
- include/
    - models/
    - external/
    - development/
    - routes/
- src/
    - services/
    - routes/
    - development/
- tests/
